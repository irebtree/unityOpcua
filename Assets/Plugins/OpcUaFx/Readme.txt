= English =
== Directory Content ==
The directory which contains this readme file contains:
* BouncyCastle.Crypto.dll - A ThirdParty assembly to work with certificates.
* Newtonsoft.Json.dll - A ThirdParty assembly used for JSON based data processing.
* Opc.UaFx.Client.deps.json - The manifest file of the dependencies of the framework.
* Opc.UaFx.Client.dll - The framework assembly from .NET Standard 2.0.
* Opc.UaFx.Client.xml - The framework IDE documentation from .NET Standard 2.0.
* System.Buffers.dll - A ThirdParty assembly used for optimized binary data processing.
* System.Management.dll - A ThirdParty assembly for system-related information.
* System.Memory.dll - A ThirdParty assembly used for optimized binary data processing.
* System.Reflection.Emit.dll - A ThirdParty assembly used to dynamically define types.
* System.ServiceModel.Primitives.dll - A ThirdParty assembly to communicate using WCF.
* unix/System.Private.ServiceModel.dll - A ThirdParty assembly required by WCF using Unix.
* win/System.Private.ServiceModel.dll - A ThirdParty assembly required by WCF using Windows.
* unix/Microsoft.Win32.Registry.dll - A ThirdParty assembly to access the system registry using Unix.
* win/Microsoft.Win32.Registry.dll - A ThirdParty assembly to access the system registry using Windows.
